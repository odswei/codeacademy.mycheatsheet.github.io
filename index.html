HTML Tags

HTML tags are used to create HTML documents and render their properties.

Name Tag Description
figure <figure> The <figure> tag specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
figcaption <figcaption> The <figcaption> tag defines a caption for a <figure> element.
video <video> The <video> tag is used to embed video content in a document, such as a movie clip or other video streams.
source <source> The <source> tag is used to specify multiple media resources for media elements, such as <video>, <audio>, and <picture>.
audio <audio> The <audio> tag is used to embed sound content in a document, such as music or other audio streams.
nav <nav> The <nav> tag defines a set of navigation links.
